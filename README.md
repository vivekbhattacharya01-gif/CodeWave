# 🔳 CodeWave — QR Code Generator

A simple, responsive **QR Code Generator** that instantly converts any URL or text into a scannable QR code.
Built with **HTML, CSS, and JavaScript** using the [goqr.me API](https://goqr.me/api/).

![CodeWave Preview](https://github.com/vivekbhattacharya01-gif/CodeWave/blob/main/CodeWave%20GIF.gif?raw=true)

---

## ✨ Features

- 🔗 Generate QR codes from any URL or text instantly
- 📱 Cross-device accessible — works on desktop, tablet & mobile
- 🎨 Clean and intuitive UI/UX design
- 🧪 Tested by 50+ users with 30% improvement in satisfaction
- ⚡ Lightweight — no frameworks, no dependencies
- 🆓 Uses a free, reliable QR code API

## ⚡ Getting Started

```bash
# Clone the repository
git clone https://github.com/vivekbhattacharya01-gif/CodeWave.git

# Go to project folder
cd CodeWave

# Open in browser
open index.html
```

> No build tools or dependencies needed — pure HTML, CSS & JS!

---

## 📡 API Reference

This project uses the **goqr.me API** (free, no key required):

```
https://api.qrserver.com/v1/create-qr-code/?size=150x150&data=YOUR_URL_HERE
```
